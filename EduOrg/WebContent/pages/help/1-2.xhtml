СОЗДАНИЕ ЗАПИСИ О ДОКУМЕНТЕ ОБ ОБРАЗОВАНИИ

Для добавления одной записи о документе об образовании необходимо нажать кнопку «Создать» (см. Рисунок 2).
В открывшемся диалоговом окне (см. Рисунок 1) необходимо заполнить все необходимые поля (все поля кроме поля «Специализация» являются обязательными для заполнения), после чего нажать кнопку «Сохранить».
Проверка корректности данных производится в автоматическом режиме по мере их заполнения, поле, содержащее некорректные данные, выделяется красной рамкой.
В случае успешного завершения операции сохранения данных, диалоговое окно «Документ об образовании» будет закрыто. Информация о новом документе об образовании будет добавлена в таблицу.


Рисунок 1. Окно создания новой записи о документе об образовании.

Для копирования записи о документе об образовании необходимо выбрать нужную запись и нажать кнопку «Копировать» (см. Рисунок 2).
В открывшемся диалоговом окне (см. Рисунок 1) надо заполнить необходимые поля (поля из блока «Информация об образовании» и тип документа об образовании будут автоматически заполнены исходя из сведений выбранной записи), после чего нажать кнопку «Сохранить».


Рисунок 2. Окно обработки информации о документах об образовании.
